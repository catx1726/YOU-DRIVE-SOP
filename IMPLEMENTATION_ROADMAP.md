# 13 步协议自动化路线图 (Implementation Roadmap)

本文档追踪 13 步生产协议中“流程节点”的物理自动化程度，作为 SOP 2.0 的任务追踪真值源。

| 节点 | 描述 | 状态 | Gap 分析 | 优先级 |
| :--- | :--- | :--- | :--- | :--- |
| **01** | 需求探索 (Brainstorm) | 已落地 | - | - |
| **03** | 分支隔离 (Branching) | 待实现 | CI 自动化分支命名合规校验 | 中 |
| **04** | 提案联动 (Propose) | 已落地 | - | - |
| **05** | 执行联动 (Apply) | 已落地 | - | - |
| **06** | TDD 强制化 (Test-Loop) | 已落地 | 自动化测试已作为 Verify 前置熔断守卫 | - |
| **07** | 安全审计 (Audit) | 部分实现 | CI 尚未校验 ops_changelog.md | 中 |
| **08** | 产出物证据 (TDD) | 已落地 | - | - |
| **09** | 规约验证 (Verify) | 已落地 | - | - |
| **10** | 资产提纯 (Distill) | 半自动 | 需与 archive 动作强制绑定 | 中 |
| **11** | 归档 (Archive) | 已落地 | manifest 自动化生成补完 | 低 |
