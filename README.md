## 🏗️ 核心逻辑概览

本项目深度集成了 **OpenSpec** (治理底座) 与 **Superpowers** (工程肌肉)，实现了智力资产的“识别-提纯-反哺”闭环。

- **Foundry (母库)**：智力的源头，存储全局 Skills 与 Patterns。
- **Workshop (车间)**：价值的产地，对齐标准执行业务开发并回馈资产。
